About This Project

This is a demonstration of a static website deployed using AWS CDK (Cloud Development Kit) with Python. The infrastructure is defined as code, making it easy to version control, test, and deploy.

The website is hosted on Amazon S3 and served through Amazon CloudFront for optimal performance and security.
